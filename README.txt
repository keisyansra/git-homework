1. Git merupakan version control system yang akan menyimpan hasil pekerjaan kita dan 
juga perubahan dalam pekerjaan kita.
2. Fungsi giit kolaborasi yaitu memungkin tim untuk mengerjakan satu projek yang 
sama tanpa terjadi benturan/tabrakan. 
3. git init: untuk membuat repositori baru, git clone: untuk membuat salinan lokal, 
git add: untuk menambahkan file atau perubahan baru dalam sebuah repositori
